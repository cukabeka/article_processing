<h1><u>article_processing 1.1.1</u></h1>

Das Addon erstellt wenn man im Backend eingeloggt ist, vier Links im Frontend 
(Bearbeitung | Metadaten | Status &auml;ndern | Abmelden) Sie k&ouml;nnen gleich ins Backend zur Bearbeitung des Artikels, der Metadaten oder 
den Status des Artikels auf Offline setzten. 

Nach der Installation kann man sich das "div" &uuml;ber die Konfiguration an die gew&uuml;nschte Position setzen.
Es ist m&ouml;glich die Texte (Bearbeitung | Metadaten | Status &auml;ndern | Abmelden) durch Standard Bilder oder durch beliebige Bilder zu ersetzen.
